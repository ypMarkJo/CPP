### 예시

```C++
try{
//예외발생 예상지역
  throw
}catch(처리할 예외의 종류 명시){
//예외처리 코드의 삽입.
}tho
```
**▲ throw에 의해 던져진 '예외 데이터'는,'예외 데이터'를 감싸는 try블록에 의해서 감지가 되어 이어서 등장하는 catch블록에 의해 처리된다.**
> try와 catch는 항상 같이 오며 중간에 다른 코드가 들어가면 안된다.
```C++
try{
//예외발생 예상지역
}
cout<<"Simple Msg"<<endl;// 컴파일 에러
catch(처리할 예외의 종류 명시){
//예외처리 코드의 삽입.
}
```
